
Задание 1.
1. Опишите класс int.
Ответ: int - встроенный тип данных, хранящий информацию о целочисленных значенияхю. В Python тип int имеет произвольную точность, т.е. не обладает ограничением на максимальное (минимальное) значение, кроме доступной памяти. К переменной типа int применимы арифметические и побитовые операции.

2. Как можно реализовать циклы в Python?
Ответ: для большинства случаев используются два цикла while и for
Синтаксис while:
while условие:
	# код выполняется когда условие == True

Синтаксис for:
1) for i in range(начальное_число_включительно, конечное_число_невключительно, шаг_цикла):
2) for i in list1:
	# здесь i - элемент списка list1

3. Как описывать и вызывать функции в python? Что будет возвращать описываемая Вами функция по умолчанию?
Ответ: функция в Python описывается с помощью ключевого слова def перед её вызовом в коде. Если у функции отсутствует return она возвращает None.

Задания 11-14.
1. Описать 5 методов строк.
Ответ: 
.upper() - переводит все символы строки в верхний регистр
.lower() - переводит все символы строки в нижний регистр
.strip() - удаляет начальные и конечные пробелы
.replace(old, new) - заменяет подстроку old на подстроку new
.split(separator) - преобразует строку в список, элементы которого разделены с помощью подстроки separator

2. Опишите особенности работы с индексами в строке на python.
Ответ: индексация элементов строки (и других массивов) как и в других ЯП начинается с 0. Если в качестве индекса передать -1, возвращаемое значение - последний элемент строки. Также с помощью конструкции string[start:stop:step] можно получить срез строки начиная с индекса start, заканчивая (не включительно) индексом stop и шагом среза step.

Задания 15-19.
1. Описать 5 методов списков.
Ответ:
.append(x) - добавляет элемент x в конец списка
.extend(list2) - добавляет в конец элементы списка list2
.insert(i, x) - вставляет элемент на позицию i
.pop() - удаляет и возвращает последний элемент списка
.remove(x) - удаляет первый элемент со значением x

2. Опишите известные Вам способы задать список.
Ответ: 
list1 = []
list2 = [1, 2, 3]
list3 = list('hello world')
list4 = [x for x in range(10)]
list5 = list(range(10)]

3. Приведите 5 способов добавить элементы в список.
Ответ:
list1.append(1)
list1.extend([1, 2, 3])
list1.insert(1, 99)
list1 += [1, 2, 3]

4. Опишите особенности работы с индексами в списке на python.
Ответ: Аналогично индексам в строках (задания 11-14 вопрос 2): начинается с 0, поддерживается отрицательная индексация, срезы. Элементы списка можно изменить, обратившись к нему по индексу.
